# Projekt Planer

## Einführung
Willkommen zum Projekt Planer! Diese Anwendung wurde entwickelt, um Teams bei der Planung und Durchführung von Projekten zu unterstützen. Mit einer benutzerfreundlichen Oberfläche bietet sie zahlreiche Funktionen, die die Teamarbeit und Produktivität verbessern.

## Funktionen
- **Aufgabenverwaltung**: Erstellen und zuweisen von Aufgaben an Teammitglieder.
- **Zeiterfassung**: Verfolgen der aufgewendeten Zeit für verschiedene Aufgaben.
- **Teamkommunikation**: Integrierte Chat- und Kommentarfunktionen für den Austausch zwischen Teammitgliedern.
- **Berichterstattung**: Erzeugung von Berichten über den Projektfortschritt und die Ressourcenauslastung.

## Technologie-Stack
- **Frontend**: React.js
- **Backend**: Node.js mit Express
- **Datenbank**: MongoDB
- **Deployment**: Docker

## Installation
1. Klone das Repository:
   ```bash
   git clone https://github.com/benutzer/projekt-planer.git
   ```
2. Installiere die Abhängigkeiten:
   ```bash
   npm install
   ```
3. Starte den Server:
   ```bash
   npm start
   ```
